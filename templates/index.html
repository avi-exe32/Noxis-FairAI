API KEY GENERATION
Terminal (cURL) 
curl -X POST https://api.noxis.ai/v1/audit \
            -H "Authorization: Bearer $NOXIS_API_KEY" \
            -H "Content-Type: multipart/form-data" \
            -F "dataset=@latest_training_data.csv" \
            -F "model=@candidate_model.pkl" \
            -F "target_label=approved" \
            -F "sensitive_attr=race"
Python (CI/CD Pipeline Check) 
import requests

            def verify_model_fairness():
                # 1. Send model to Noxis for audit
                response = requests.post(
                    "https://api.noxis.ai/v1/audit",
                    headers={"Authorization": f"Bearer {API_KEY}"},
                    files={"dataset": open("data.csv", "rb"), "model": open("model.pkl", "rb")},
                    data={"sensitive_attr": "gender", "label_col": "hired"}
                )
                
                audit = response.json()
                
                # 2. Block deployment if Disparate Impact is illegal
                if audit['disparate_impact'] < 0.8:
                    raise Exception("DEPLOYMENT BLOCKED: Model violates EEOC 4/5ths rule.")
                    
                print("Model passes fairness checks. Proceeding to production.")
